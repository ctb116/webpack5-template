## Description

Project template using webpack 5. See webpack's [Getting Started Guide](https://webpack.js.org/guides/getting-started/)

Configuration includes:

- style-loader and css-loader
- asset/resource
  - images/files
  - fonts

- development mode
- devtool: inline-source-map
- webpack-dev-server

- HtmlWbepackPlugin
